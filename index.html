Anthropic Launches Code Review for Claude Code to Tame AI-Generated Code

Anthropic released Code Review, a multi-agent tool that automatically reviews pull requests for logical errors in AI-generated code. Targeted at enterprise customers like Uber, Salesforce, and Accenture, it integrates with GitHub and uses parallel agents to catch bugs before they ship. Each review costs $15–$25 on average, reflecting the resource-intensive multi-agent architecture.

OpenAI Acquires Promptfoo to Bake Security Testing into Its Enterprise Platform

OpenAI is acquiring Promptfoo, an AI security testing platform used by 25% of Fortune 500 companies. The technology will be integrated into OpenAI's Frontier enterprise platform, adding automated testing for prompt injections, jailbreaks, and data leaks. Promptfoo had raised $23 million at an $86 million valuation; its open-source project will continue post-acquisition.
